============================
The ``5-text_indentation`` module
============================

Using ``text_indentation``
--------------------

Import module
    >>> text_indentation = __import__('5-text_indentation').text_indentation

======
String
Print a text, changing '.' , '?' and ':' per '\n'
======

    >>> text = "Human. is. a crack"
    >>> text_indentation(text)
    Human.
    <BLANKLINE>
    is.
    <BLANKLINE>
    a crack

    >>> text = "Human: is: a crack"
    >>> text_indentation(text)
    Human:
    <BLANKLINE>
    is:
    <BLANKLINE>
    a crack

    >>> text = "Human? is? a crack"
    >>> text_indentation(text)
    Human?
    <BLANKLINE>
    is?
    <BLANKLINE>
    a crack

    >>> text = "Human. is: a? crack"
    >>> text_indentation(text)
    Human.
    <BLANKLINE>
    is:
    <BLANKLINE>
    a?
    <BLANKLINE>
    crack
